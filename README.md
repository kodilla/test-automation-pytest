# Calculator app

This is a simple calculator app that can add, subtract, multiply and divide.
It works only with integers.

## Usage

```
$ python3 main.py <left_number> <operation> <right_number>
```

## Examples

```bash
$ python3 main.py 1 + 3
4
$ python3 main.py 1 - 3
-2
$ python3 main.py 2 x 3
6
$ python3 main.py 6 / 3
2
```

## Running tests

```bash
$ pytest
```
